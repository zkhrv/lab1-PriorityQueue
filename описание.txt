метод "init" инициализирует объект класса PriorityQueue. Он устанавливае начальное 
значение размера очереди равное 0 и создает пустой список для хранения элементов.

Метод "add_element" добавляет новый элемент в очередь. Элемент добавляется в конец 
списка, затем происходит перестройка очереди, чтобы сохранить свойство кучи. 

Метод "rewrite" перестраивает кучу с корнем в узле с индексом левого и правого потомка, 
затем выбирается максимальный из потомков и, если он больше узла с индексом "max", 
то значения узлов меняются местами. После этого происходит рекурсивный 
вызов метода "rewrite" для перестройки кучи дальше вниз по дереву.

Метод "del_max_el" удаляет максимальный элемент из очереди (который всегда находится 
в корне кучи), затем перестраивает кучу, чтобы сохранить свойства кучи.

Метод "del_element" удаляет заданный элемент "del_el" из очереди, если он 
существует в ней, и перестраивает очередь после удаления элемента. Если 
элемент не найден в очереди, выводится сообщение об этом.

Метод "get_max_el" возвращает максимальный элемент в очереди, то есть первый элемент массива "arr".

Метод "count" выводит количество элементов в очереди.

Метод "write" выводит на экран содержимое очереди.
